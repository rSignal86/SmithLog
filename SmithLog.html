How to use

  1. Load a Cabrillo .txt file (or paste its content), then press SPACE to Analyze.
  2. Optionally set Contest (dropdown), Comment, and TX Power.
  3. Press F10 to Download the ADIF file and import it into QRZ.

Notes

  • Frequency strings are preserved exactly as in Cabrillo.
  • Contest serials are auto-mapped to STX/SRX when numeric; exchange text is kept in STX_STRING/SRX_STRING.
  • Selecting CQ-WW in the dropdown adds MY_CQ_ZONE/CQZ from the exchange.
