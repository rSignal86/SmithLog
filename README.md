# SmithLog
Forge your contest logs into ADIF log to upload where ever you want.
SmithLog is a lightweight, web-based tool designed for amateur radio operators to convert contest logs from Cabrillo format into ADIF files ready for QRZ and other logging platforms.

## ✅ Features

Cabrillo → ADIF Conversion
QRZ-Compatible

## Optional Parameters
Add a global comment and TX power to all QSOs.
Dynamic Contest Dropdown to log Your QSO's as a contest at QRZ


## 🔍 How It Works

1.	Upload or paste your Cabrillo log.
2. Select the contest from the dropdown.
3. Optionally add TX power, global comment and type of contest.
4. Click Generate ADIF and download the file.
